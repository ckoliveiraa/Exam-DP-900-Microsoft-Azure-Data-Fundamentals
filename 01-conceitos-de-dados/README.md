# Módulo 1 — Conceitos Centrais de Dados

> **Domínio 1 do exame DP-900** — Peso: **25–30%**

Este módulo apresenta os fundamentos de dados: como os dados são representados, onde são armazenados, os tipos de carga de trabalho (transacional e analítica) e os papéis profissionais envolvidos.

---

## Objetivos de Aprendizagem

Ao final deste módulo, você será capaz de:

- Diferenciar dados **estruturados, semiestruturados e não estruturados**.
- Reconhecer os principais **formatos de arquivo** de dados.
- Identificar **opções de armazenamento** de dados no Azure.
- Distinguir cargas de trabalho **transacionais (OLTP)** de **analíticas (OLAP)**.
- Descrever as **funções e responsabilidades** de DBA, engenheiro de dados e analista de dados.
