# 1.6 — Funções e Responsabilidades em Dados

> Trabalhar com dados envolve diferentes profissionais, cada um com um foco distinto. O exame DP-900 cobra **três funções principais**: **administrador de banco de dados**, **engenheiro de dados** e **analista de dados**.

---

## Administrador de Banco de Dados (DBA)

Responsável pela **saúde do banco de dados** ao longo de todo o ciclo de vida.

**Responsabilidades:**

- Projetar, **instalar e configurar** bancos de dados.
- Garantir **disponibilidade** e **desempenho** (otimização de consultas, índices).
- Planejar e executar **backup e restauração**; testar a recuperação de desastres.
- Gerenciar **segurança**: usuários, permissões, criptografia.
- Monitorar o banco e aplicar **atualizações**.

**Ferramentas típicas:** SQL Server Management Studio (SSMS), Azure Data Studio, Portal do Azure.

---

## Engenheiro de Dados (Data Engineer)

Responsável por **construir e manter a infraestrutura de dados** — colocar os dados certos no lugar certo, no formato certo.

**Responsabilidades:**

- Projetar e operar **pipelines de dados** (ingestão, integração, transformação).
- Implementar processos de **ETL/ELT**.
- Integrar dados de **múltiplas fontes** (bancos, arquivos, APIs, streaming).
- Garantir **qualidade**, consistência e **governança** dos dados.
- Cuidar da **privacidade e segurança** dos dados em repouso e em trânsito.

**Ferramentas típicas:** Azure Data Factory, Azure Synapse Analytics, Azure Databricks, Microsoft Fabric.

---

## Analista de Dados (Data Analyst)

Responsável por **transformar dados em informação útil** para o negócio.

**Responsabilidades:**

- **Explorar e modelar** os dados para identificar padrões e tendências.
- Criar **relatórios, painéis e visualizações**.
- Aplicar cálculos e medidas para responder a perguntas de negócio.
- Comunicar **insights** que apoiem a tomada de decisão.

**Ferramentas típicas:** Microsoft Power BI, Excel.

---

## Resumo Comparativo

| Função | Foco principal | Pergunta que responde | Ferramenta típica |
|--------|----------------|------------------------|-------------------|
| **DBA** | Manter o banco seguro, disponível e rápido | "O banco está saudável?" | SSMS, Portal do Azure |
| **Engenheiro de dados** | Construir e manter pipelines de dados | "Os dados estão prontos e no lugar certo?" | Data Factory, Synapse, Databricks |
| **Analista de dados** | Transformar dados em insights visuais | "O que os dados estão dizendo?" | Power BI |

---

## Como o Exame Cobra

As questões costumam **descrever uma tarefa** e pedir a função responsável. Exemplos:

- "Configurar um backup automático" → **DBA**.
- "Criar um pipeline que carrega dados de várias fontes" → **engenheiro de dados**.
- "Montar um dashboard de vendas no Power BI" → **analista de dados**.

---

## Pontos-Chave para o Exame

- ✅ **DBA** → instalação, segurança, backup, desempenho e acesso ao banco.
- ✅ **Engenheiro de dados** → pipelines, ETL/ELT e integração de dados de várias fontes.
- ✅ **Analista de dados** → relatórios, visualizações e insights de negócio.
- ✅ **Power BI** está associado ao **analista**; **Data Factory** ao **engenheiro**.
- ✅ Saiba **associar a tarefa à função** correta — formato comum de questão.
