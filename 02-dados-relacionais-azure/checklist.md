# Checklist de Revisão — Módulo 2 (Dados Relacionais no Azure)

## 2.1 Conceitos de Dados Relacionais
- [ ] Sei o que são tabela, linha e coluna
- [ ] Diferencio **chave primária** de **chave estrangeira**
- [ ] Entendo os três tipos de integridade (entidade, referencial, domínio)
- [ ] Sei que cada coluna tem um **tipo de dado** definido

## 2.2 Normalização
- [ ] Explico o que é normalização e **por que** ela é usada
- [ ] Entendo o problema da redundância e da anomalia de atualização
- [ ] Sei a relação entre normalização e **junções** na leitura
- [ ] Associo OLTP a normalizado e OLAP a desnormalizado

## 2.3 SQL — Comandos Comuns
- [ ] Memorizei **DDL, DML e DCL** e seus comandos
- [ ] Sei o efeito de `SELECT`, `INSERT`, `UPDATE`, `DELETE`
- [ ] Lembro que `UPDATE`/`DELETE` sem `WHERE` afetam tudo
- [ ] Sei que **T-SQL** é o dialeto do Azure SQL

## 2.4 Objetos de Banco de Dados
- [ ] Sei que uma **view** é consulta salva e não armazena dados
- [ ] Entendo o que é um **stored procedure**
- [ ] Conheço a vantagem e o custo de um **índice**
- [ ] Diferencio índice **clustered** (um por tabela) de **non-clustered**

## 2.5 Família Azure SQL
- [ ] Diferencio **IaaS × PaaS**
- [ ] Conheço os 3 serviços: SQL Database, Managed Instance, SQL Server em VM
- [ ] Sei qual usar para **migração** (Managed Instance) e para **apps novos** (SQL Database)
- [ ] Sei o que é um **pool elástico**

## 2.6 Bancos Open-Source no Azure
- [ ] Conheço os serviços para **MySQL, PostgreSQL e MariaDB**
- [ ] Sei o que o Azure gerencia em um serviço PaaS
- [ ] Lembro que o **MariaDB** está em descontinuação

## Revisão Final do Módulo
- [ ] Refiz os **flashcards** sem erros
- [ ] Atingi **≥ 80%** no simulado do módulo
- [ ] Revisei o **glossário**
