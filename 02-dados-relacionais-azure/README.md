# Módulo 2 — Dados Relacionais no Azure

> **Domínio 2 do exame DP-900** — Peso: **20–25%**

Este módulo cobre os conceitos de bancos de dados relacionais e os serviços relacionais oferecidos pelo Azure.

---

## Objetivos de Aprendizagem

Ao final deste módulo, você será capaz de:

- Explicar as características de **dados relacionais** e a **normalização**.
- Reconhecer os principais comandos **SQL** (DDL, DML, DQL).
- Identificar **objetos de banco de dados** (tabelas, views, procedures, índices).
- Descrever a **família Azure SQL** (SQL Database, Managed Instance, SQL Server em VM).
- Descrever os serviços **open-source no Azure** (MySQL, PostgreSQL, MariaDB).
