# Módulo 4 — Cargas de Trabalho de Análise no Azure

> **Domínio 4 do exame DP-900** — Peso: **25–30%**

Este módulo cobre análise em larga escala, análise em tempo real e visualização de dados com o Power BI.

---

## Objetivos de Aprendizagem

Ao final deste módulo, você será capaz de:

- Descrever os elementos de uma **análise em larga escala** (ingestão e processamento).
- Diferenciar **data warehouse, data lake e lakehouse**.
- Identificar os serviços de análise do Azure: **Synapse Analytics, Databricks, Microsoft Fabric, Data Factory**.
- Distinguir processamento em **lote (batch)** e em **fluxo (streaming)**.
- Descrever as capacidades de **visualização de dados no Microsoft Power BI**.
