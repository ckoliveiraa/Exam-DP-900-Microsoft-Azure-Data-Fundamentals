# 4.5 — Visualização de Dados com Power BI

> O **Microsoft Power BI** é a plataforma de **business intelligence** da Microsoft para transformar dados em **relatórios e painéis** interativos que apoiam a tomada de decisão.

---

## Componentes do Power BI

| Componente | O que é |
|------------|---------|
| **Power BI Desktop** | Aplicativo (Windows) para **criar** modelos, relatórios e visualizações. |
| **Serviço do Power BI** | Plataforma online (SaaS) para **publicar, compartilhar** e consumir conteúdo. |
| **Power BI Mobile** | Aplicativo para visualizar relatórios e painéis em celulares e tablets. |

> Fluxo típico: **criar** no Power BI Desktop → **publicar** no Serviço → **consumir** na Web ou no Mobile.

---

## Modelo de Dados

Antes de visualizar, os dados são organizados em um **modelo**:

- Reúne **tabelas** importadas de uma ou várias fontes.
- As tabelas são ligadas por **relacionamentos**.
- É possível criar **medidas** e **colunas calculadas** com a linguagem **DAX**.
- Um bom modelo costuma seguir o **esquema estrela** (fatos + dimensões).

---

## Relatórios × Painéis (Dashboards)

| | Relatório (*report*) | Painel (*dashboard*) |
|---|---|---|
| Páginas | Pode ter **várias** | Uma **única** página |
| Origem | Um único conjunto de dados | Pode reunir visuais de **vários** relatórios |
| Interatividade | Alta (filtros, drill-down) | Visão consolidada de destaques |
| Onde é criado | Power BI Desktop ou Serviço | Apenas no Serviço do Power BI |

---

## Escolher a Visualização Certa

Cada tipo de dado pede um visual adequado:

| Objetivo | Visualização indicada |
|----------|------------------------|
| Comparar valores entre categorias | Gráfico de **colunas/barras** |
| Mostrar tendência ao longo do tempo | Gráfico de **linhas** |
| Mostrar proporção de um todo | Gráfico de **pizza/rosca** |
| Exibir um único indicador (KPI) | **Cartão** ou visual de KPI |
| Distribuição geográfica | **Mapa** |
| Dados detalhados linha a linha | **Tabela** ou **matriz** |

---

## Pontos-Chave para o Exame

- ✅ **Power BI Desktop** = criar; **Serviço do Power BI** = publicar/compartilhar; **Mobile** = consumir.
- ✅ O **modelo de dados** reúne tabelas com relacionamentos; cálculos usam **DAX**.
- ✅ **Relatório** = várias páginas, um conjunto de dados; **painel** = uma página, vários relatórios.
- ✅ Saiba **escolher a visualização** certa: linhas para tendências, colunas para comparações, mapa para geografia.
