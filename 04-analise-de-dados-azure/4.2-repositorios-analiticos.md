# 4.2 — Repositórios Analíticos: Warehouse, Lake, Lakehouse

> Depois de ingeridos e processados, os dados precisam de um **repositório analítico**. Há três modelos principais — cada um com um equilíbrio diferente entre estrutura e flexibilidade.

---

## Data Warehouse

Um repositório de dados **estruturados**, modelado e otimizado para **consultas analíticas**.

- Os dados chegam já **limpos, transformados e modelados** (geralmente em esquema estrela).
- Esquema definido **antes** da gravação (*schema-on-write*).
- Excelente desempenho para relatórios e BI sobre dados estruturados.
- Consultado com **SQL**.

---

## Data Lake

Um repositório que armazena **grandes volumes de dados brutos** em qualquer formato — estruturados, semiestruturados e não estruturados.

- Os dados são guardados **como estão**, sem transformação prévia.
- Esquema aplicado **na leitura** (*schema-on-read*).
- Baixo custo e altamente escalável.
- No Azure, é construído sobre o **Azure Data Lake Storage Gen2**.

---

## Data Lakehouse

Um modelo **híbrido** que combina o melhor dos dois:

- A **flexibilidade e o baixo custo** de um data lake.
- A **estrutura, o desempenho e a confiabilidade** de um data warehouse.
- Permite armazenar dados brutos e, ao mesmo tempo, consultá-los com desempenho de warehouse.
- É a abordagem adotada pelo **Azure Databricks** e pelo **Microsoft Fabric**.

---

## Comparativo

| | Data Warehouse | Data Lake | Lakehouse |
|---|---|---|---|
| Tipo de dado | Estruturado | Qualquer tipo | Qualquer tipo |
| Esquema | Na gravação | Na leitura | Combina os dois |
| Custo | Maior | Menor | Intermediário |
| Desempenho analítico | Alto | Variável | Alto |
| Dados brutos | Não | Sim | Sim |

---

## Pontos-Chave para o Exame

- ✅ **Data warehouse** = dados **estruturados**, modelados, *schema-on-write*, ótimo para BI.
- ✅ **Data lake** = dados **brutos de qualquer tipo**, *schema-on-read*, baixo custo.
- ✅ **Lakehouse** = **híbrido** — flexibilidade do lake + desempenho do warehouse.
- ✅ O data lake no Azure usa o **Data Lake Storage Gen2**.
