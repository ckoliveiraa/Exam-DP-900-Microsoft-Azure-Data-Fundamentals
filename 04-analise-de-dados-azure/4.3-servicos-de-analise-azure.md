# 4.3 — Serviços de Análise do Azure

> O Azure oferece um conjunto de serviços para construir soluções de análise em larga escala. Conhecer **o papel de cada um** é essencial para o exame.

---

## Azure Data Factory

- Serviço de **integração de dados** baseado em **pipelines**.
- Faz a **ingestão e a transformação** (ETL/ELT) de dados de muitas fontes.
- Não armazena nem analisa os dados — ele os **move e prepara**.

## Azure Synapse Analytics

- Plataforma de análise **unificada** que reúne data warehouse, big data e integração de dados.
- Combina **SQL**, **Apache Spark** e pipelines em um só ambiente.
- Voltada a soluções de análise corporativa de ponta a ponta.

## Azure Databricks

- Plataforma de análise baseada em **Apache Spark**, otimizada para o Azure.
- Forte em **engenharia de dados**, **machine learning** e processamento de big data.
- Adota a arquitetura **lakehouse**.

## Azure HDInsight

- Serviço gerenciado de **frameworks open-source de big data** (Hadoop, Spark, Kafka, HBase).
- Útil para quem já usa esses ecossistemas open-source.

## Microsoft Fabric

- Plataforma de análise **tudo em um** (SaaS), que **unifica** ingestão, data lake, data warehouse, ciência de dados, análise em tempo real e Power BI.
- Usa um data lake central chamado **OneLake**.
- É a direção mais recente da Microsoft para análise de dados.

---

## Resumo dos Papéis

| Serviço | Papel principal |
|---------|-----------------|
| **Azure Data Factory** | Ingestão e transformação (pipelines ETL/ELT) |
| **Azure Synapse Analytics** | Análise unificada (warehouse + Spark + pipelines) |
| **Azure Databricks** | Análise e ML com Apache Spark (lakehouse) |
| **Azure HDInsight** | Frameworks open-source de big data gerenciados |
| **Microsoft Fabric** | Plataforma SaaS tudo-em-um, com o OneLake |

---

## Pontos-Chave para o Exame

- ✅ **Data Factory** = ingestão e transformação via **pipelines**; não analisa dados.
- ✅ **Synapse Analytics** = análise **unificada** (SQL + Spark + pipelines).
- ✅ **Databricks** = análise e **machine learning** com **Apache Spark**.
- ✅ **Microsoft Fabric** = plataforma **SaaS tudo-em-um**, com o data lake **OneLake**.
- ✅ Saiba **associar o serviço ao seu papel** — formato comum de questão.
