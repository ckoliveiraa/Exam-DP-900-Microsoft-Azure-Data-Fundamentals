## Sobre o Exame

| Item | Detalhe |
|------|---------|
| **Código** | DP-900 |
| **Certificação** | Microsoft Certified: Azure Data Fundamentals |
| **Formato** | Múltipla escolha e múltipla resposta |
| **Questões** | ~40–60 (Microsoft não divulga o número exato) |
| **Duração** | 45 min para concluir a avaliação |
| **Nota de corte** | 700 / 1000 |
| **Custo** | ~USD 59 — varia conforme o país/região onde a prova é aplicada |
| **Idiomas** | Inglês, Português (Brasil), Espanhol, Francês, Alemão, Italiano, Japonês, Coreano, Russo, Árabe, Indonésio, Chinês (Simplificado e Tradicional) |
| **Validade** | Não expira (certificação Fundamentals) |
| **Nível** | Iniciante (Fundamentals) |

## Pré-requisitos

- Conta Azure ativa (free tier recomendada)
- Noções básicas de TI e de dados (opcional, mas útil)

## Como Usar Este Curso

1. Leia os módulos na ordem.
2. Execute os labs práticos sugeridos no portal do Azure.
3. Revise **flashcards** e **glossário** ao final de cada módulo.
4. Faça o **simulado** de cada domínio (meta: ≥ 80%).
5. Complete o [**simulado final interativo**](https://ckoliveiraa.github.io/Exam-DP-900-Microsoft-Azure-Data-Fundamentals/) antes de agendar o exame.
