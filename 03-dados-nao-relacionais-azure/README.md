# Módulo 3 — Dados Não Relacionais no Azure

> **Domínio 3 do exame DP-900** — Peso: **15–20%**

Este módulo cobre o armazenamento não relacional do Azure e o Azure Cosmos DB.

---

## Objetivos de Aprendizagem

Ao final deste módulo, você será capaz de:

- Descrever as capacidades do **Armazenamento do Azure** (Blob, File e Table).
- Identificar os **níveis de acesso** do Blob Storage.
- Descrever as capacidades e casos de uso do **Azure Cosmos DB**.
- Reconhecer as principais **APIs do Cosmos DB**.
